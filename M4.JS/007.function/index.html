Hàm trong js

Hàm trong js là 1 tập hợp gồm nhiều câu lệnh được gom nhóm lại thành 1 module để thực hiện những tính năng cụ thể

Hàm tính chu vi hình tròn

Hàm dùng tính chu vi hay diện tích hình chữ nhật

Hàm dùng để chuyển độ C thành độ F

Hàm Chuyển đổi dặm thành km

    Cú pháp :
    function tên_hàm(tham_số_1,tham_số_2,tham_số_3 ...) {
        // code thực thi của hàm
    }
    Gọi hàm :
    tên_hàm(gia_tri_1, giá_trị_2 ...);
    Hoặc
    tên_hàm(bieesn1, biến 2 ...)

Xây dựng 2 function

  1. tính chu vi hình tròn
  2. tính diện tích hình tròn
