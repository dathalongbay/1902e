Vòng lặp là quá trình lặp đi lặp lại
    có bước bắt đầu
    có bước kết thúc
    và nó sự thay đổi sau mỗi vòng lặp
    1 - for
    2 - whilte
    3 - do .. while
    Cú pháp for :
    for(bước khởi đầu; điều kiện vòng lặp chạy; sự thay đổi sau mỗi vòng lặp) {
        // code thực thi trong vòng lặp
    }

    Cú pháp while :

    bước khởi đầu của vòng lặp while
    while(điều kiện vòng lặp chạy) {
        // code thực thi trong vòng lặp

        sự thay đổi sau mỗi vòng lặp nhớ đặt ở dòng cuối của while
    }

    Cú pháp do ... while

    khởi đầu của vòng lặp
    do {
        // code thực thi trong vòng lặp

        sự thay đổi sau mỗi vòng lặp ( nên đặt ở dòng cuối của do )
    }while(Điều kiện để vòng lặp chạy);

    *** sự khác biệt giữa vòng lặp while và do while
    nếu điều kiện đúng thì cả 2 vòng lặp đều chạy
    nhưng lần đầu tiên cho dù điều kiện sai thì vòng lặp do while nó vẫn chạy lần đầu tiên
    còn vòng lặp while chỉ chạy khi điều kiện đúng
    do ... while sẽ luôn chạy lần đầu tiên bất kể điều kiện đúng hay sai
    từ lần thứ 2 trở đi nó mới check điều kiện để đảm bảo chỉ chạy khi mà điều kiện là true
