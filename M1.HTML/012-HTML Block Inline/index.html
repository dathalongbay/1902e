HTML BLOCK Và INLINE

Block có nghĩa 1 khối

Inline có nghĩa là 1 hàng

Trong html các thẻ chia thành 2 kiểu hiển thị chính là inline và block

Block chứa độ rộng toàn màn hinh

Inline chứa độ rộng tương ứng với nội dung bên trong
